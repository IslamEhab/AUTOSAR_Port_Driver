Detailed Description

Used to hold register with their offsets by saving them in the struct with their order from data sheet.

Note
BSRR divided into 2 registers BSRRL (Least significant 16 bit) which used to set certain bit with '1' & BSRRH (Most significant 16 bit) Which used to clear certain bit
